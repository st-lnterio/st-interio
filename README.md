# ST Interio Website

A free GitHub Pages website inspired by your company presentation video.

## Publish on GitHub Pages
1. Create a new repository on GitHub.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and the **/ (root)** folder.
6. Save. Your website will be live in a few minutes at:
   `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

## Customize
- Edit `index.html` to add your real phone number and email.
- Replace images inside the `assets` folder with your latest project photos.
